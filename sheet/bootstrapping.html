⇦Django CheatSheet

First of all install the django-crispy-form using following command.

	pip install django-crispy-forms

Open the settings.py file from project app and add the following code in INSTALLED_APP sorounding double cotation mark.

	crispy_forms

Also add the following snippet after INSTALLED_APP

	CRISPY_TEMPLATE_PACK = 'bootstrap4'

Now re-write your form.html file with the following snippet in form element after csrf_token.

	{% load crispy_forms_tags %}
	{{ form.as_p | crispy"}}

We also add crispy form in different field of a form. Follow the below snippet


    
{{ form.district|as_crispy_field  }}

    
{{ form.thana|as_crispy_field  }}
